Mahabharata: A Data-Driven Analysis of Influence, Allegiance, and Fate

A portfolio project showcasing end-to-end data analysis, modeling, and dashboard design in Power BI.

Interactive Dashboard Walkthrough

Project Summary & Key Accomplishments

Developed a comprehensive, multi-page Power BI dashboard to analyze the 50 key characters of the Mahabharata, transforming qualitative narrative data into measurable, interactive insights on character influence, factional strength, and ultimate fate.

  • Data Transformation (ETL): Cleaned and transformed raw, unstructured text in Power Query; used conditional logic to group diverse entries (e.g., "Killed by Bhima") into a single "Killed in Battle" category and standardize various royal titles into a clean "Role" column.
  • Data Modeling & DAX: Resolved data integrity issues by using DISTINCT COUNT measures, ensuring 100% accuracy in all KPIs and visuals after row-splitting transformations (like splitting role) had created duplicate entries.
  • Interactive Dashboard & UX Design: Architected an intuitive, multi-page dashboard with drill-through capabilities; built a custom navigation system by linking a "View Profile" button to a slicer, replacing the default right-click menu to create a more user-friendly experience.
